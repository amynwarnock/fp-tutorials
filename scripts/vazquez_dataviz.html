Introduction

This tutorial will walk through an applied example to show how to simulate survey data and create a function to graph survey data using divergent bar charts. Creating this function would allow the user to graph survey data to any of their liking!

Simulating data

We begin by simulating some survey data. We will simulate responses to 5 questions that use a likert item with a scale of one to four. To ease the simulation of the data, we will create a function to generate this data. The function has 5 formals or arguments: 1. max - An integer that represents the maximum of the likert scale. Since the items we are developing are on a 4 point scale, then we will add 4. If we wanted to simulate items on a 7 point scale, then we would add a 7 to this argument. 2. n - An integer that represents the total number observations that the user wishes to generate 3. weights - A list of weights that the user specifies to influence the frequency of certain responses. The length of the vector should be equal to the max. 4. levels - A numeric vector that lists the numeric number tied to each response. It should start with 1 and end with the max number specified. 5. labels - A character vector with the labels attached to the numeric values.

The function below also provides an error message in case certain formals are not specified correctly.

likert_sim <- function(max, n, weights, levels,labels) {
  if ((length(levels) == length(labels)) & (length(weights) == max))  {
  factor(sample(1:max, n, replace = TRUE, prob = weights),
         levels = levels,
         labels = labels)
  } else {
  stop("Are the length of your labels, levels, or weights equal to the likert item specified? ",  
          " The scale is on a ", max, " point likert item scale. Did you add sufficient arguments?  ",
         ", Number of weight arguments added was ",length(weights),
         ", Number of labels arguments added was ", length(labels),
         ", Number of level arguments added was, ", length(levels))
  }
}

summ_likert <- function(df,mincol,maxcol,labels,levels) {
  df %>%
  pivot_longer(
    cols = {{mincol}}:{{maxcol}},
    names_to = "question",
    values_to = "frequency") %>%
  group_by(question,frequency) %>%
    count(name = "n_answers") %>%
  group_by(question) %>%
  mutate(percent_answers = (n_answers / sum(n_answers))*100,
         frequency = factor(frequency,
                           levels = levels,
                           labels = labels))
}

graph_likert <- function(df) {
  df %>%
  ggplot(aes(x = question,
             y = percent_answers,
             fill = frequency)) + 
  geom_bar(stat = "identity") +
  geom_text(aes(label = abs(percent_answers)),
            position = position_stack(vjust = 0.5),
            color = "white",
            fontface = "bold") +
  coord_flip() +
  scale_x_discrete() +
  scale_fill_viridis_d() +
  labs(title = "To what extent do you agree with the following statements?",
       x = "Percent of respondents",
       fill = NULL) +
  theme_minimal() +
  theme(axis.text.x = element_blank(),
        axis.title.x = element_blank(),
        panel.grid = element_blank(),
        legend.position = "bottom")
}
